# Uplift Modeling Project

Проект решает три задачи маркетингового ML: uplift-моделирование, контактная политика, propensity-моделирование и кластеризация.

## Описание

- **Uplift-моделирование**: Оценка прироста конверсии от маркетинговых предложений.
- **Контактная политика**: Правила коммуникации (Активно, Стандарт, Пауза).
- **Propensity-моделирование и кластеризация**: Склонность к покупке и сегментация клиентов.

Датасет: https://www.kaggle.com/datasets/davinwijaya/customer-retention
